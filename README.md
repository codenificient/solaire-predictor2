# Solaire Predictor

A solar energy prediction model using Python and machine learning to forecast solar panel output.

## Tech Stack

- Python
- Machine Learning (scikit-learn)
- Docker
- UV package manager

## Getting Started

```bash
uv sync
python main.py
```

## Docker

```bash
docker build -t solaire-predictor .
docker run solaire-predictor
```
